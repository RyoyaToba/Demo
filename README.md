## Description

このリポジトリは、Java と Spring Boot を使用したアプリケーションの作成を練習するためのリポジトリです。
個人利用のみを目的としており、サーバー上に公開されることはありません。使用する場合はローカル環境で行ってください。
仕事が忙しいのであまり更新しないかもしれません。

## Environment Setup

application.ymlはリポジトリにあげていませんので、resources配下にご自身の環境で作成ください。
